    How to Fix Nil Error During Setting Sass Hugo

    March 21, 2021
    Hugo, Blog, coding

    Error #

    index.html:6:46": execute of template failed: template: index.html:6:46: executing “index.html” at <resources.ToCSS>: error calling ToCSS: type not supported in Resource transformations

    It was a bit time wasting. but the asset folder must be created in root folder.

    Solution #

    then the following lines worked.

    {{ $sass := resources.Get "sass/main.scss" }}
{{ $style := $sass | resources.ToCSS }}
<link rel="stylesheet" href="{{ $style.Permalink }}">

    Attention #

    There is another asset folder in resource/_gen. Don’t confused with it! You must create new asset folder in root directory.
